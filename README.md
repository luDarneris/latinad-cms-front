# LatinAd CMS Frontend

Repositorio para el ejercicio tecnico de LatinAd. Incluye la API mock provista para el desafio y una base frontend en AngularJS 1.x.

## Estructura

```txt
api-mock/   API local provista para el ejercicio
frontend/   Aplicacion frontend AngularJS
```

## Requisitos

- Node.js 18 o superior
- npm 9 o superior

## Ejecutar API mock

```bash
cd api-mock
npm install
npm run dev
```

La API queda disponible en:

```txt
http://localhost:3001
```

Credenciales demo:

```txt
Usuario: admin
Password: admin123
```

## Ejecutar frontend

En otra terminal:

```bash
cd frontend
npm install
npm run dev
```

## Stack frontend

- AngularJS 1.8
- Vite
